# maixduino-ili9341
Maixduino (Kendryte K210 AI accelerator) with ILI9341-based TFT display

### Wiring Maixduino - ILI9341**
- 12        - SDO/MISO
- 3V3       - LED
- 13        - SCK
- 11        - SDI/MOSI
- 8         - DC/RS
- 9         - RESET
- 7         - CS
- GND       - GND
- 5V        - VCC

### Known Issues
Very slow refresh rate... only 4 fps.
